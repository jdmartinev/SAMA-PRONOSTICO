# SAMA Pronóstico

**Sistema de Alerta y Monitoreo de Antioquia**
Departamento Administrativo de Gestión del Riesgo — DAGRAN
---

Este repositorio consolida el trabajo desarrollado en conjunto con el equipo de
pronóstico de SAMA. Incluirá modelos, pipelines de datos, utilidades de evaluación
y documentación técnica asociada al componente de pronóstico de precipitación del
sistema de alerta temprana de Antioquia.

> Repositorio en construcción. El contenido se irá incorporando a medida que
> avance el trabajo con el equipo.
